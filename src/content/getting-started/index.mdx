## Prerequisites

Before getting started, ensure you have:

- **Node.js** 16.x or later
- **npm** or **yarn** package manager
- A modern web browser
- Basic knowledge of JavaScript/TypeScript

## Quick Installation

Choose your preferred installation method:

### Option 1: CLI Tool (Recommended)

```bash
# Install globally
npm install -g @arten/cli

# Navigate to your project
cd my-nextjs-app

# Start Arten (auto-detects project type)
arten start
```

### Option 2: Standalone Application

```bash
git clone https://github.com/your-username/arten.git
cd arten
npm install
echo "OPENROUTER_API_KEY=your_api_key_here" > .env.local
npm run dev
```

## Your First Test

Once Arten is running, you can create your first test:

1. **Open the web interface** at `http://localhost:3000`
2. **Describe your test** in natural language:
   ```
   Test the login form with valid credentials
   ```
3. **Generate the test** using AI
4. **Review and customize** the generated test
5. **Run the test** across multiple browsers
